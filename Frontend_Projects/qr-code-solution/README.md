I learned Three things by doing this project.
1. Basic CSS
2. CSS variable Usage
3. Centering ``` div ``` 

Usage of CSS variable Code Snippet:
```css

    --fontname: "Outfit", sans-serif;
}
.title {
    font-family: var(--fontname);
}
```
